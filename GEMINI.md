# Istruzioni di Progetto

- Dopo ogni modifica al codice che sia stata verificata con successo, esegui sempre `git add`, `git commit` (con un messaggio descrittivo) e `git push`.
- Il push attiva automaticamente il deploy su Vercel, quindi assicurati che il codice sia stabile prima di inviarlo.
- Se mancano variabili d'ambiente nel file `.env.local`, avvisa l'utente.
